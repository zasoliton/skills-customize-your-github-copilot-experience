
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman-style word guessing game that uses Python strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Game Logic

#### Description
Create the core game mechanics for a Hangman-style word guessing experience.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list
- Track guessed letters and display current progress using `_ _ _` formatting
- Count incorrect guesses and stop after a limited number of attempts
- End the game when the player guesses the word or runs out of attempts

### 🛠️ User Interaction

#### Description
Add input/output handling so the player can make guesses and see the game state.

#### Requirements
Completed program should:

- Prompt the player for a letter guess each round
- Show letters already guessed and the remaining attempts
- Print a win message when the player guesses the full word
- Print a lose message and reveal the secret word when attempts are exhausted
